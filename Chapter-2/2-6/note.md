`r => real value`

| &            | \|            | ^             |
| ------------ | ------------- | ------------- |
| `1 & r => r` | `1 \| r => 1` | `1 ^ r => 0`  |
| `0 & r => 0` | `0 \| r => r` | `0 ^ r => ~r` |



## one
如果没有`(unsigned)`，无论右移几位输出皆为`ffffffff`。
```c
printf("x: %x\n", ((unsigned)(~0) >> 4));
```
